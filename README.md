# 👩‍💼 Gestion des Employés

L’application **WorkUnity** est une plateforme web créée avec Laravel pour faciliter la gestion des informations du personnel. Elle permet à l’administrateur d’ajouter, modifier, supprimer ou consulter les employés, tandis que chaque employé peut accéder à ses propres informations.

---

## 🎯 Objectifs du projet

- 📋 Afficher la liste des employés  
- ➕ Ajouter un employé  
- ✏️ Modifier un employé  
- 🗑️ Supprimer un employé  
- 🧪 Préparer la base pour des fonctionnalités futures  

---

## 👤 User Stories

- **US1** – Consultation de profil : voir mes informations personnelles  
- **US2** – Ajouter un employé (nom, prénom, email, poste)  
- **US3** – Modifier un employé existant  
- **US4** – Supprimer un employé  
- **US5** – Lister tous les employés  
- **US6** – Préremplir la base avec des employés fictifs via Factory & Seeder
